>	This is a demo about navigationBar and it can be used as a hot-plug component. 


## Vesrion 1.0.1

This version requires Xcode 8.0.

## Setup with CocoaPods

```

pod 'ZXScrollNavigationBar', '~> 0.0.1'

use_frameworks!

```
